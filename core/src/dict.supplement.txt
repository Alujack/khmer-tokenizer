# Supplemental Khmer dictionary for khmer-tokenizer-core.

# Project-authored wordlist (MIT OR Apache-2.0, same as the crate) covering
# modern vocabulary absent from the bundled chamkho/SIL base dictionary
# (`dict.txt`, frozen 2015): province names, countries, loanwords, and
# technology terms. Every entry was confirmed missing from `dict.txt` and
# hand-verified against real-world usage before inclusion — candidates were
# *ranked* by frequency of appearance in public Khmer web-news text, but the
# list itself is original curation, not an extract of any corpus.
#
# One word per line. Blank lines and lines starting with '#' are ignored.
# See core/ATTRIBUTION.md ("Supplemental wordlist").
